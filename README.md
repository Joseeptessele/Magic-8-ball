# Magic-8-ball

An IOs application to simulate the Magic 8 Ball toy, where the user ask a yes-or-no question to the ball and it answars back.

# What I Learned

* IBOutlets & IBActions ✅
* Variables, Constants e Data types ✅
* Randomisation ✅
* Arrays ✅
* Functions & Motion detection ✅
